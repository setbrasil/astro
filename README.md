## 📦 Instalação

Certifique-se que tem o [Go](https://golang.org/dl/) instalado, depois:

```bash
git clone https://github.com/seu-usuario/astro.git
cd astro
go build -o astro
```

Opcionalmente, mova o binário para o PATH:

```bash
mv astro /usr/local/bin/
```

---

## 🖥️ Comandos

### `astro start`

Inicia o web server.

```bash
astro start
```
